1. Always-comes rules

  • Instruments: Clean → Disinfect → Sterilize
  • Recent transplant / unstable systemic → defer elective
  • Asthma analgesia → paracetamol (caution NSAID)
  • Gingival enlargement → nifedipine / phenytoin / cyclosporine
  • Implant safety to IAN ≈ 2 mm
  • New crown food packing → open contact
  • Rotary file break → torsional (tip lock) ± flexural
  • Narrowest canal = apical constriction
  • Pulp cap / apexification → MTA / bioceramic
  • Avulsion splint ~ 2 weeks flexible (IADT)
  • Kennedy I = bilateral distal extension
  • Worst RPD force on abutment = horizontal/lateral
  • White patch can’t wipe = leukoplakia (clinical)
  • Thrush membrane wipes off
  • Pierre Robin = micrognathia + glossoptosis + cleft
  • Natal = at birth · Neonatal ≤ 30 days
  • Diastema + frenum → often ortho then frenectomy
  • Zygoma # → diplopia
  • Odontogenic infection flora = mixed
  • MRONJ → bisphosphonates / denosumab
  • COPD → semi-supine
  • Dam wrinkled between teeth → holes too far apart

2. Restorative ~40% HIGHEST WEIGHT

Operative

  • Microfill: polish ↑ strength ↓ · fracture → hybrid/nanohybrid
  • Rubber dam isolation troubleshooting
  • Direct pulp cap: MTA/bioceramic when indicated
  • Full crown prep = most reduction / pulp risk
  • Fluoride → remineralization / fluorapatite-like resistance

Fixed

  • Food packing: open contact / emergence / occlusion
  • Ferrule, retention grooves, short crown options
  • Finish lines: chamfer/rounded shoulder ceramics
  • Cements: match substrate (resin / RMGI / GI)

RPD / CD

  • Kennedy I–IV; distal extension support (mucosa+teeth)
  • Lingual bar needs depth; else plate
  • Clasp: retention/reciprocation/encirclement
  • CD: CR, VDO, sore spots, reline concepts

Implant basics

  • ≥2 mm from IAN · platform / bone-level concepts
  • Mucositis (soft) vs peri-implantitis (+ bone loss)
  • Uncontrolled DM → prefer removable over implants

3. Periodontics ~18%

  • 2017: Stage = severity/complexity · Grade = rate/risk
  • CAL from CEJ → base
  • Furcation: I incipient · II cul-de-sac · III through · IV visible
  • Plaque = primary gingivitis etiology
  • Smoking: worse disease, masked bleeding, poor healing
  • Drug enlargement: CCB / phenytoin / cyclosporine
  • Biological width violation → chronic inflammation
  • SRP first-line cause-related therapy
  • Peri-implant disease continuum
  • Diabetes ↔ perio bidirectional risk

4. Endodontics ~17%

  • Lingering spontaneous pain → irreversible pulpitis pattern
  • Pus / sinus → necrosis / chronic apical abscess path
  • WL near apical constriction
  • NaOCl = organic · EDTA = smear inorganic · don’t mix CHX+NaOCl
  • Upper molar persistent RL → think untreated MB2
  • Torsional vs flexural file failure
  • Retrieval: ultrasonics / removal kits (case-based)
  • VRF: isolated deep pocket, J-shaped, post-endo tooth
  • Apexification MTA barrier; trauma per IADT
  • Avulsion: storage medium + time critical

5. Oral med & surgery ~15%

  • Dry socket 2–4 d: irrigate, dressing, analgesia
  • IANB failures: anatomy / height / accessory innervation
  • Lidocaine with epi classic max ~7 mg/kg (know concept)
  • Epi caution: hyperthyroid / some cardiac
  • Warfarin simple XO: often continue if INR OK + local hemostasis
  • Biopsy: incisional large/suspect · excisional small · FNA salivary
  • Dentigerous: enucleation / marsupialization large
  • MRONJ risk drugs — prevent traumatic extractions when possible
  • Spaces & mixed flora odontogenic infections
  • ZMC: diplopia, malar flat, V2 paresthesia

6. Ortho / Pedo ~10%

  • Class II = lower molar distal · Div2 retroclined U1 + deep bite
  • Class III skeletal adult: often ortho+surgery pathways
  • Expansion: Haas / Hyrax / quad helix (age/need)
  • Space: band & loop · distal shoe if 6 unerupted
  • SSC: multi-surface caries / post pulp therapy primary molars
  • Fluoride supplement: only if water low + risk — not if optimal
  • Behavior: Tell-Show-Do first
  • Natal vs neonatal · Pierre Robin triad

7. Ethics · IC · medical

  • Barriers / disinfect light handles every patient · hand hygiene #1
  • COPD semi-supine · pregnancy: elective defer 1st trimester often; pyogenic granuloma observe if OK
  • Endocarditis prophylaxis: know principles (who truly needs)
  • Needle-stick: wash, report, protocol
