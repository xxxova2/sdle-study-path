B. Kennedy classification I–IV (with examples)

B1. Core rule before the four boxes

  • Kennedy classification describes the arch pattern of edentulous areas for RPD design language. It is not a prognosis score and not a material choice by itself.
  • Classification is based on the most posterior edentulous area (Applegate). Name the class from the worst posterior problem first, then count modifications.
  • Bounded span = teeth present both anterior and posterior to the edentulous space (tooth‑supported potential). Free‑end / distal extension = no posterior abutment distal to the saddle — base ends on soft tissue.
  • Always classify the arch you will actually restore after planned extractions (Applegate) — not the arch from last year’s chart if teeth are coming out next week.

B2. Kennedy Class I — bilateral distal extension

  • Definition: edentulous areas bilateral posterior to remaining natural teeth — free‑end saddles on both sides.
  • Example: remaining anteriors and premolars both sides; all molars missing both left and right. No terminal molar abutments for the distal ends of either saddle.
  • Clinical flag: support is mixed (tooth anteriorly + mucosa under free‑end base). Highest classic teaching concern for saddle sink, abutment torque, need for stress‑control clasp concepts, maximum base extension, indirect retention.
  • Memory: “I = both free ends” — two posterior orphans.

B3. Kennedy Class II — unilateral distal extension

  • Definition: edentulous area unilateral posterior to remaining natural teeth — free‑end on one side only.
  • Example: missing lower left molars with no left terminal abutment, while the right side still has a molar present that bounds that side (or is intact). One free‑end saddle; the other side is not a free‑end of the same type.
  • Clinical flag: same distal‑extension biomechanics as Class I on the free‑end side; cross‑arch design and major connector rigidity still matter to share force.
  • Memory: “II = one free end.”

B4. Kennedy Class III — unilateral bounded span

  • Definition: unilateral edentulous area with natural teeth anterior and posterior to it — a bounded saddle on that side (tooth‑supported if abutments sound).
  • Example: missing 4 and 5 with 3 and 6 present (or missing a single premolar/molar with neighbors both sides). Classic “tooth‑bounded” partial logic.
  • Clinical flag: better support potential from teeth at both ends of the span; less classic free‑end torque story than I/II, but still needs rests, retention, reciprocation, and rigid major connector.
  • Memory: “III = bounded side (tooth each end of the gap).”

B5. Kennedy Class IV — anterior crossing midline

  • Definition: a single edentulous area anterior to remaining posterior teeth that crosses the midline.
  • Example: missing upper centrals and laterals (or more) with canines or posterior teeth remaining bilaterally — one continuous anterior gap that crosses the midline.
  • Clinical flag: esthetics + phonetics + often challenging retention without over‑relying on unsightly clasps; still tooth‑bounded posteriorly if posteriors remain, but design is its own family of stems.
  • Memory: “IV = front gap that crosses the middle.”

B6. Modification spaces

  • Modification = additional edentulous area(s) besides the space that determined the class. Number of modifications = number of extra edentulous areas (not how big they are).
  • Example language: Class II mod 1 = unilateral free‑end plus one other edentulous area elsewhere; Class III mod 1 = bounded main span plus one other gap.
  • Extent of the modification is not graded in the class name — only the count of additional areas (classic Applegate teaching).
  • Class IV handling of “modifications” is a classic exam trap zone — see Applegate section; spirit of Class IV is the single anterior midline‑crossing space as the defining feature.

B7. Exam stem patterns (Kennedy)

  • “Bilateral free‑end saddles” / “missing all posterior teeth both sides with anterior teeth remaining” → Class I.
  • “Free‑end only on the left; right has terminal molar” → Class II.
  • “Missing 15 and 16 with 14 and 17 present” (bounded) → Class III on that side.
  • “Missing 11–22, canines present, posteriors present” → Class IV.
  • Stem lists two gaps: decide which is most posterior for class, then count the other as modification if rules allow.
  • Do not reclassify a Class I as Class III just because “there are still some teeth left.” Free ends rule the class when they are the most posterior spaces.

C. Applegate rules (high yield, not trivia)

C1. Rules banks actually test

  • Classify after extractions that are part of the treatment plan — the class is for the arch you will restore, not the temporary pre‑extraction state if teeth are condemned.
  • If a third molar is missing and not to be replaced, it is not considered in classification (does not create a “space” that changes class naming).
  • If a third molar is present and used as an abutment, it is considered.
  • If a second molar is missing and not to be replaced, it is not considered (classic Applegate item — do not invent a free‑end purely from an unrestored second molar you will never replace).
  • The most posterior edentulous area determines the class.
  • Other edentulous areas are modification spaces, designated by their number (mod 1, mod 2…).
  • Extent of modification is ignored in the name — only how many additional areas.
  • Class IV has no modification spaces that keep it Class IV in the classic rule set in the way students sometimes invent — additional posterior spaces change how you think about classification; know the spirit: Class IV = single anterior edentulous area crossing the midline.

C2. Why Applegate exists on the exam

  • Without rules, two candidates class the same mouth differently and design arguments collapse. The exam uses Applegate so there is a shared naming language.
  • Common student error: counting every missing tooth as a separate modification, or classifying before planned extractions, or treating non‑replaced second/third molars as free‑end class changers when Applegate says not to consider them.
  • Another error: naming Class III when the most posterior problem is actually a free‑end — always re‑check the most posterior space.

C3. Exam stem patterns (Applegate)

  • “Which determines Kennedy class?” → most posterior edentulous area.
  • “Third molar missing, not replaced” → ignore for classification.
  • “Third molar present, used as abutment” → count it.
  • “Second molar missing, not to be replaced” → not considered.
  • “Classification performed before planned extractions” → wrong timing; classify the planned arch.
  • “How are additional spaces named?” → modifications by number, not size.

D. Support, stability, retention, reciprocation + distal extension + RPI

D1. Four force words (must be cold)

  • Support = resistance to tissueward (vertical seating) forces. Primary sources: occlusal/cingulum/incisal rests on teeth; for free‑end bases also the residual ridge mucosa/bone under the saddle. Without support, the prosthesis sinks and abuses tissues and abutments.
  • Stability = resistance to horizontal / lateral / anteroposterior shifting (rocking, sliding). Guide planes, minor connectors, major connector rigidity, denture base flanges, and bracing arms contribute.
  • Retention = resistance to forces that try to lift the prosthesis away from the tissues (sticky foods, gravity on upper, clasp undercuts, adhesion/cohesion/atmospheric pressure for complete dentures, guide plane friction).
  • Reciprocation = a rigid element that counters the lateral force of a flexible retentive clasp arm during insertion and removal so the abutment is not torqued in only one direction. Bracing/reciprocal arm or plate concepts — “clasp push without a counter = tooth abuse.”

D2. Which force is most destructive on abutments?

  • Many teaching banks emphasize horizontal / lateral forces and torque on abutment teeth (especially distal extension abutments) as highly destructive compared with well‑distributed vertical load through rests.
  • A passive clasp at rest that only engages undercut under controlled flexure is the ideal story; continuous hard pressure = periodontal abuse.
  • Encirclement rule (classic): clasp assembly should engage more than 180° of the abutment circumference in appropriate design so the tooth is not pried away from a single contact point.

D3. Distal extension (Kennedy I and II) — the biomechanical problem

  • In Class I/II, the free‑end saddle sits on compressible mucosa. Under load the saddle can sink; a rigid connection to a distal abutment can then torque that abutment (lever action).
  • Support contrast: tooth‑supported Class III can load through rests at both ends of the span; free‑end support is tooth + mucosa, and mucosa yields more than the PDL — differential movement is the design problem.
  • Design responses banks expect you to recognize: broad coverage of the residual ridge (maximum extension within anatomic limits), accurate impression of the supporting tissues (altered cast / functional concepts where taught), mesial rests on terminal abutments in many stress‑release systems, indirect retainers (rests anterior to the fulcrum line) to resist rotation of the free‑end base away from the ridge, and stress‑releasing clasp assemblies such as RPI.
  • Indirect retention: when sticky food pulls the free‑end base up, the prosthesis rotates around a fulcrum line through the terminal abutments; a rest anterior to that fulcrum resists the rotation (classic “indirect retainer” definition).

D4. RPI clasp concept (classic distal extension)

  • RPI = Rest + Proximal plate + I‑bar. Memorize parts and roles — free points.
  • R — mesial rest: rest seat on the mesial of the abutment (not a heavy distal rest that locks a fulcrum right at the free‑end side in the classic RPI story). Provides support and helps the rotation geometry under load.
  • P — proximal plate: contacts the distal guide plane of the abutment; contributes guiding, reciprocation/bracing roles in the assembly, and helps control path of insertion. As the free‑end sinks, design intent is favorable movement relative to the abutment.
  • I — I‑bar: infrabulge (bar) retentive clasp approaching from below the survey line, engaging a mid‑buccal (or designed) undercut. Flexible tip provides retention; the approach arm is designed for stress release philosophy as the saddle loads.
  • Exam gold: RPI on distal extension abutments is the named system that signals “stress control / reduce torque” thinking. If stem says mesial rest + distal proximal plate + I‑bar → RPI.
  • Related names sometimes appear: RPA (rest, proximal plate, Akers/wrought wire variants depending on school) — still know RPI cold first.

D5. Clasp and rest basics that travel with RPI

  • Rest function = support (vertical) and helps maintain clasp–tooth relationship; rest seats prepared in enamel/restoration must have proper floor incline so force is directed favorably along the long axis as much as design allows.
  • Retentive arm = flexible terminal third engages undercut; body more rigid.
  • Reciprocal / bracing arm = rigid, contacts tooth ideally before or as retentive tip flexes (reciprocation timing concept).
  • Surveyor logic: path of insertion, undercut gauge, tripoding — stems may say “0.01 inch undercut” language; concept is measured undercut for the clasp type, not random bend of wire in the mouth.
  • Passivity at rest: assembly should not continuously spring‑load the abutment when the prosthesis is fully seated without functional load.

D6. Combination syndrome flags (know the picture)

  • Combination syndrome (Kelly): classic pattern when a patient has an edentulous maxilla opposing a Kennedy Class I mandibular partial (only anterior lower teeth remaining) — often with a complete upper denture.
  • Flags/findings teaching lists: bone loss / flabby anterior maxillary ridge, overgrowth of maxillary tuberosities, papillary hyperplasia of hard palate, extrusion of lower anterior teeth, loss of bone under mandibular distal extension bases, and occlusal plane / VDO problems from the destructive cycle of anterior hyperfunction.
  • Exam use: recognize the scenario and that poorly managed complete upper vs lower free‑end partial can drive this pattern — prevention/planning matters (occlusion scheme, base support, recall, possible implant support concepts in modern care).

D7. Exam stem patterns (forces / distal extension / RPI)

  • “Which component resists tissueward movement?” → support / rests (and ridge for free‑end).
  • “Which resists lifting away from tissues?” → retention.
  • “Clasp flexes against abutment with no rigid counter” → missing reciprocation.
  • “Distal extension base sinks, terminal abutment torqued” → free‑end mucosa compression + leverage; design with RPI‑type thinking, base extension, indirect retention.
  • “Name the parts of RPI” → mesial rest, proximal plate, I‑bar.
  • “Rest location in classic RPI” → mesial.
  • “Indirect retainer location” → rest anterior to fulcrum line of free‑end.
  • “Edentulous upper denture + lower bilateral free‑end partial, flabby anterior maxilla…” → combination syndrome flags.

E. Major connectors (maxilla / mandible)

E1. Job of the major connector

  • Major connector joins components of one side of the arch to the other and must be rigid. Flexibility concentrates force on abutments and soft tissues and destroys the “one prosthesis” idea.
  • It must not impinge on movable tissue improperly, must be compatible with speech and tongue space when possible, and must allow hygiene (food traps under poorly designed connectors are classic complaints).
  • Rigidity rule (exam gold): major connector failure theme #1 = too flexible / inadequate bulk or poor design choice for the span.

E2. Maxillary major connectors (names + when)

  • Full / complete palatal plate (or coverage): maximum rigidity and support distribution; used when many teeth missing, long spans, need for strength, or when other connectors would be inadequate. Respects posterior border / vibrating line concepts when coverage is extensive; can feel bulkier to patients.
  • Anterior–posterior palatal strap (A‑P strap): two straps with an open mid‑palate area; good rigidity for many bilateral designs when anatomy allows; common “strong without full coverage” answer family.
  • Palatal strap (single wide strap): for intermediate situations; must be wide enough for rigidity — a skinny strap flexes.
  • U‑shaped / horseshoe: open mid‑palate, follows alveolar ridges — often the least rigid of the common options when used for long spans; preferred mainly when torus or patient factors force mid‑palate open, or for some anterior replacements — banks often treat horseshoe as weaker/less preferred when a strap/plate would work.
  • Maxillary connector borders: avoid movable soft palate improperly; finish edges smooth; anterior border usually posterior to rugae when possible for speech/comfort in many designs (school nuances exist — know rigidity + tissue compatibility as the testable core).

E3. Mandibular major connectors

  • Lingual bar: preferred when there is adequate vertical space from gingival margins / free gingival tissues to the movable floor of mouth. Classic teaching cites needing enough clearance (often ≈7–8 mm total space concepts appear in notes — memorize “enough depth for bar” rather than fighting over 1 mm). Half‑pear shape, superior border several mm from gingiva.
  • Lingual plate (linguopolate): when insufficient depth for a bar, high floor of mouth, anticipated future anterior tooth loss (can add clasps/rests later more easily), periodontally weakened anteriors needing stabilization/plating as indicated, or when more rigidity / indirect retention help is wanted.
  • Sublingual bar / continuous bar / labial bar: less common primary answers; labial bar when severe lingual undercuts or tori block lingual approaches — know they exist for odd stems.
  • Exam picker: inadequate lingual vestibule / high floor → lingual plate, not force a bar that impinges on the floor when the tongue lifts.

E4. Minor connectors and tissue relief (brief)

  • Minor connectors join rests/clasps/bases to the major connector; they should be rigid, located in embrasures appropriately, and not create food traps against soft tissue without relief where required.
  • Relief under major connectors over soft tissue (mandible especially) prevents settling impingement as the metal framework is seated and as tissues change — stems about “blanching ridges under bar” point to inadequate relief or wrong border position.

E5. Exam stem patterns (major connectors)

  • “Most important property of major connector” → rigidity.
  • “Not enough space from gingiva to floor of mouth” → lingual plate.
  • “Adequate space, prefer less tissue coverage” → lingual bar.
  • “Long span maxillary bilateral free ends, need max strength” → full palate / broad rigid design family.
  • “Horseshoe chosen for long distal extensions without torus” → often the wrong choice vs more rigid straps/plates in teaching stems.
  • “Metal flexes across midline under load” → inadequate major connector rigidity.

F. Complete denture — CR, VDO, phonetics, post dam, sore spots

F1. Centric relation vs centric occlusion (CD language)

  • Centric relation (CR): a bone‑to‑bone / ligament‑guided maxillomandibular relationship independent of tooth contact — the reference position used heavily in complete denture construction because there are no stable natural tooth‑defined contacts left.
  • Centric occlusion (CO) / MIP in natural dentitions is tooth‑guided maximum intercuspation. In complete dentures, artificial teeth are set so that occlusion is harmonious with the recorded CR in classic teaching (CR = CO developed for the dentures).
  • Exam contrast: CR is not “where the teeth hit hardest in the old dentures” if those teeth were wrong. CR is the jaw relation record foundation; CO is the tooth contact scheme you build.
  • Recording CR: patient relaxed, proper guidance technique (school methods vary: chin point, bimanual, etc.) — exam cares that CD occlusion is organized around a repeatable CR record, not random bite of collapsed old dentures alone.

F2. Vertical dimension of occlusion (VDO)

  • VDO = face height when teeth (or denture teeth) are in occlusion. Related: VDR (rest) and interocclusal rest space (freeway space) = VDR − VDO concepts.
  • VDO too high (overopened): muscle strain / fatigue, difficulty closing, sore ridges from constant pressure, clicking of teeth during speech, facial strain, possible TMJ discomfort, patient feels “teeth keep hitting,” phonetic trouble, gagging sensation sometimes from bulk.
  • VDO too low (collapsed): aged / collapsed face appearance, angular cheilitis risk from fold maceration, reduced chewing efficiency, tongue/cheek biting patterns, possible TMJ strain patterns, patient may look overclosed with deep folds.
  • Clinical checks: rest face height, phonetics, appearance, freeway space comfort — not a single magic millimeter for every face.

F3. Phonetics as a clinical instrument

  • Speech tests help verify tooth position and VDO. Classic sounds: sibilants (“s”, “z”, “ch”, “j”, “sh”) and fricatives; “f” / “v” for maxillary incisal edge against wet line of lower lip (labiodental) in many teaching setups.
  • Whistle / lisp / “s” becoming “sh” can flag incorrect vertical dimension, incorrect anterior tooth set (too far labial/lingual), or incorrect speaking space.
  • Exam use: phonetics is a check, not the only record. Still free points when stem ties speech complaint to VDO or tooth position.

F4. Post dam / posterior palatal seal

  • Post dam (posterior palatal seal): intentional slight compression of the denture base into the compressible tissue at the junction of hard and soft palate along the vibrating line area to complete the posterior seal of the maxillary complete denture.
  • Purpose = retention of the upper denture (atmospheric seal / border seal completion posteriorly). Without it, air leaks, denture drops on speech or function.
  • Vibrating line / Ah line concepts locate the posterior border; seal is developed in impression / cast scoring / processing depending on technique — exam cares about function: seal + correct location, not lab gossip.
  • Over‑deep post dam → tissue injury / sore line; under‑developed → leak and drop. Poor seal differential also includes overextended/underextended flanges and occlusion lifting the base.

F5. Sore spots and border problems

  • Sore spots after insertion: usually overextension of flanges, pressure spots on the intaglio (nodules, sharp acrylic, undercuts trapped), premature occlusal contacts, or sharp residual ridge / spicule under the base.
  • Method: mark with pressure‑indicating paste (PIP) or disclosing media, relieve carefully, recheck occlusion — do not grind randomly forever without diagnosis.
  • Overextended borders → muscle action dislodges denture (stability/retention fail) + sore vestibules.
  • Underextended borders → poor stability/retention, food under base, less support from flanges.
  • Neutral zone / flange form must respect tongue, lips, cheeks — stability is a muscle‑molded story as much as a suction story.
  • Frenum notches: inadequate relief → sore frenum and dislodging; over‑relief → seal leak.

F6. Insertion‑day checklist (stem gold)

  • Evaluate intaglio pressure spots → adjust. Evaluate borders in function → adjust overextensions. Evaluate occlusion in CR/CO developed → eliminate prematurities. Evaluate phonetics and VDO appearance. Instruct patient on insertion, care, expected adaptation, and recall.
  • Day‑1 post‑insert pain that is sharp on a point = often base pressure; bilateral muscle fatigue after hours = often VDO or occlusion scheme issues — still diagnose, do not guess.

G. Relining vs rebasing

G1. Definitions (one‑line each)

  • Reline: resurface the tissue side of the denture base with new material so the intaglio fits the residual ridge again — teeth and most of the polished base bulk remain. Used when fit is poor (ridge resorption / base warpage) but teeth, occlusion scheme, and VDO are still acceptable.
  • Rebase: replace most or all of the denture base material, keep the teeth (classic concept) — when the base is porous, stained, fractured beyond simple repair, or deficient but the teeth are still good and position is salvageable.
  • New denture: when teeth are worn flat, VDO collapsed, esthetics hopeless, occlusion plane destroyed, or base + teeth both beyond salvage — not endless reline theater.

G2. Clinical sequence rules

  • Tissues must be healthy before final impression for a definitive reline. Treat abused tissues first (tissue conditioners, leave denture out part‑time, adjust overextensions, correct gross occlusion) — classic “reline on inflamed flabby abused ridge without conditioning” is a fail path.
  • Chairside soft liners / temporary relines are interim tools; laboratory processed hard relines are the durable story when indicated.
  • Check VDO and centric after reline — adding material can alter vertical if technique is sloppy.
  • If patient needs “reline” but presents with collapsed VDO and worn teeth, the honest plan is often remake, not only a tissue surface refresh.

G3. Exam stem patterns (reline / rebase)

  • “Denture loose, teeth and occlusion OK, ridge resorbed” → reline (after tissue health).
  • “Base stained/porous/deficient, teeth still good” → rebase family.
  • “Teeth worn, VDO collapsed, poor esthetics” → new denture, not infinite reline.
  • “Immediate reline on red swollen tissues” → condition tissues first.

H. Impression materials comparison (table in bullet form)

H1. How to read any materials stem

  • Ask three questions: (1) What is being impressed? (preliminary diagnostic, final fixed, final edentulous, border mold). (2) What property kills or saves the case? (stability over time, wettability, stiffness/undercuts, tear strength, pour timing). (3) What is the classic bank pairing? (alginate → pour soon; PVS → final stable; polyether → wet + stiff; ZOE → edentulous mucostatic, no undercuts).

H2. Alginate (irreversible hydrocolloid)

  • Use: preliminary impressions, study models, opposing casts, orthodontic models, many removable prelims — cheap, fast, decent surface for non‑critical work.
  • Strengths: hydrophilic, easy mix, inexpensive, acceptable for many diagnostics.
  • Weaknesses: poor long‑term dimensional stability (syneresis/imbibition stories) — pour soon; tears more easily than elastomers; not the first choice for precision fixed finals when PVS/polyether available.
  • Exam gold: “alginate impression left on tray overnight” → distortion. Pour promptly; store briefly in 100% humidity if delay is unavoidable (still not ideal long delay).

H3. PVS / addition silicone (polyvinyl siloxane)

  • Use: definitive fixed impressions, many final removable impressions, bite registrations (specific products) — workhorse final elastomer.
  • Strengths: excellent dimensional stability, accurate detail, multiple pours often acceptable, pleasant relative to older rubbers, good elastic recovery.
  • Weaknesses: historically hydrophobic (modern hydrophilized/surfactants help but dry field still matters); sulfur inhibition stories with some latex gloves on putty; cost higher than alginate; must wait full set before removal.
  • Exam gold: precision crown/bridge final → PVS (or polyether) not alginate alone.

H4. Polyether

  • Use: definitive impressions when accuracy + wettability desired; good detail in slightly moist fields (classic teaching strength vs older hydrophobics).
  • Strengths: accurate, hydrophilic / good wettability story, good for multiple abutments.
  • Weaknesses: stiff / rigid set — can lock in severe undercuts, harder to remove, risk of cast/tooth trauma if forcibly removed; some products sensitive to storage/water uptake historically.
  • Exam gold: wet mouth + need accuracy → polyether often; severe undercuts → caution / prefer more flexible elastomer.

H5. Polysulfide (rubber base) — still in older banks

  • Long working time historically, messy, odor (mercaptan), custom tray often, pour within a reasonable time (less stable long‑term than PVS). Appears as “which is true of polysulfide” comparison stems more than as modern first choice.

H6. Zinc oxide eugenol (ZOE) impression paste

  • Use: classic mucostatic final impressions for edentulous ridges (stable, non‑undercut ridges) in complete denture techniques; also used historically as wash in custom trays.
  • Strengths: rigid, good surface detail on appropriate ridges, dimensionally stable for its use case.
  • Weaknesses: rigid — locks/breaks in undercuts; eugenol can irritate some soft tissues; not for dentate undercut impressions.
  • Exam gold: edentulous final mucostatic + no undercuts → ZOE paste family; undercuts present → not ZOE paste.

H7. Impression compound and others

  • Impression compound: thermoplastic; border molding and preliminary edentulous handling; reheated and adapted; not a fine secondary detail material alone for modern fixed work.
  • Agar (reversible hydrocolloid): historical accurate wet‑field material needing water‑cooled trays/equipment — rare modern clinical but can appear in comparison tables.
  • Digital scan: same clinical rules as Day 2 — dry visible margin, soft tissue management — not a free pass to ignore blood and saliva.

H8. Picker logic cheat sheet

  • Preliminary / study / opposing → alginate (often).
  • Final fixed precision → PVS or polyether.
  • Wet field wettability emphasis → polyether often.
  • Severe undercuts + very stiff material → avoid polyether / rigid pastes.
  • Pour ASAP / left on bench hours → alginate problem.
  • Edentulous mucostatic no undercut → ZOE paste classic.
  • Border mold → compound (or heavy body PVS border techniques in modern workflows).

H9. Exam stem patterns (impressions)

  • “Impression poured 24 h later, distorted cast, alginate used” → delayed pour / alginate instability.
  • “Best dimensional stability for delayed pour” → PVS (addition silicone) classic answer family.
  • “Accurate but difficult to remove from undercuts” → polyether stiffness.
  • “Mucostatic edentulous final, firm ridge, no undercuts” → ZOE.
  • “Undercut mandibular ridge, rigid paste locked” → wrong material choice.
  • “Crown final, alginate only” → inadequate for precision fixed in modern bank logic.

I. Gypsum types + acrylic resins

I1. Gypsum types I–V (uses)

  • Type I — impression plaster: historic impression material (rare modern clinical). Weak, for impressions of edentulous ridges in old techniques.
  • Type II — model plaster: study models, mounting, articulator mounts where high strength not critical; higher water need, weaker, whiter “plaster” feel.
  • Type III — dental stone (Class I stone / microstone family): working casts for dentures, many orthodontic casts, general lab casts — moderate strength.
  • Type IV — high‑strength low‑expansion stone (die stone): dies and precision fixed work — high strength, low expansion so die dimensions stay accurate. Exam gold: crown/bridge dies → Type IV.
  • Type V — high‑strength high‑expansion stone: high strength with higher setting expansion used in some compensatory techniques (e.g., when expansion is desired to offset other shrinkages in certain lab workflows). Strength like die stones but expansion profile differs from Type IV.

I2. Water/powder and handling rules

  • More water than recommended → weaker cast, more porosity, generally more setting expansion — do not “soup” stone to make pouring easier.
  • Less water → thicker mix, may not flow into detail, trapped voids, harder manipulation.
  • Vacuum mixing / vibration reduces bubbles. Wet the impression properly for hydrocolloids per technique; follow manufacturer W/P ratio — free points.
  • Separate cast at appropriate time; premature separation damages surface; endless soak can affect surface hardness depending on material.

I3. Acrylic (PMMA) denture bases — heat‑cure vs self‑cure

  • Heat‑cured (heat‑polymerized) PMMA: common for definitive denture bases. Powder (polymer) + liquid (monomer); packed in flask; controlled heat curing cycle. Generally better strength / lower residual monomer when processed correctly than slapdash self‑cure.
  • Self‑cure / auto‑polymerizing / cold‑cure: chemical initiator systems; used for repairs, relines, orthodontic appliances, some temporary bases. Convenient but classically higher residual monomer and often lower strength / more porosity risk if handled poorly.
  • Light‑cured acrylic systems exist for certain lab/chair applications — still process per product.
  • Residual monomer can irritate tissues if processing is incomplete; proper cure reduces free monomer.

I4. Porosity causes (classic exam traps)

  • Gaseous porosity: monomer boils when temperature rises too fast / too high before polymer matrix is ready — classic “hot fast cure” exam line. Vapor voids inside thick sections (palate center classic).
  • Inadequate pressure / underpacking: not enough dough packed into the flask → voids.
  • Improper mixing / too much monomer: liquid‑rich mix → porosity / weakness patterns.
  • Rapid heating early in cycle: same boiling story — follow manufacturer curing schedule (slow heat‑up for thick bases).
  • Consequences of porosity: weak base, stain, odor, hygiene problems, fracture risk, residual monomer pockets.
  • Warpage risk also rises with improper processing and deflasking while too hot — stems about ill‑fitting processed denture after lab error.

I5. Other acrylic / processing free points

  • Denture teeth may be acrylic or porcelain historically — acrylic bonds chemically to base more readily; porcelain needed mechanical retention features and can click / wear opposing.
  • Repair fractures: approximate fragments, use appropriate repair acrylic, reinforce if design allows — but repeated midline fractures of upper denture may signal occlusion / VDO / midline stress / thin palate problems needing remake redesign.
  • Soft liners are not permanent solutions for every loose denture; diagnose why the denture is loose (resorption, occlusion, borders, VDO).

I6. Exam stem patterns (gypsum / acrylic)

  • “Die stone for crown dies” → Type IV.
  • “Study model only” → Type II plaster often acceptable.
  • “Denture cast working model” → Type III stone family often.
  • “Extra water in mix” → weaker, more expansion/porosity risk.
  • “Bubbles in thick palatal acrylic after boil‑out cure rushed” → gaseous porosity / overheating monomer.
  • “Repair of broken flange chairside” → self‑cure acrylic family.
  • “Definitive base best processed how?” → controlled heat‑cure cycle (not random microwave folklore unless stem specifies a system).
